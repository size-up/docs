# Summary

Configuration references and useful files and links used by `Bare Metal Server`, `Virtual Private Server (VPS)`, `Virtual Machine (VM)` and other `Cloud` stuff.

---

## Kubernetes section:

Is a `Kubernetes` configuration repository. It contains `Kubernetes` configuration files and `Kubernetes` configuration files for `Kubernetes` components.

[Kubernetes](https://kubernetes.io/) is a distributed, fault-tolerant, and highly-scalable software platform for automating the deployment, configuration, and ongoing management of containerized applications.

## k3s section:

All about k3s. k3s is a fork of kubernetes. It is a lightweight container runtime for Kubernetes. For more information, follow this link: [k3s.io](https://k3s.io/).

## Ansible section:

Located in `/ansible/`. [Ansible](https://www.ansible.com/) is a tool for automating deployment, configuration management, and application orchestration.

There is also playbooks examples in `/ansible/playbooks/`<br/>
[Ansible Playbook](https://docs.ansible.com/ansible/latest/intro_playbooks.html). It is a text file that contains a series of commands that are executed on a remote host.

## Virtual Machine section:

All configuration files and links related to `Virtual Machine (VM)` and `Virtual Private Server (VPS)` stuff. E.g. `cloud-init`, `.profile` files, etc.
